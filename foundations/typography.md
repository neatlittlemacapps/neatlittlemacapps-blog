# Typography

Typography creates hierarchy and improves readability across our products.

## Font Families

**Primary Font: Inter**
- Used for: UI text, body copy, headings
- Fallback: `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`

**Monospace Font: JetBrains Mono**
- Used for: Code blocks, technical content
- Fallback: `"Courier New", monospace`

## Type Scale

| Style | Size | Line Height | Weight | Usage |
|-------|------|-------------|--------|-------|
| H1 | 32px | 1.2 | 700 | Page titles |
| H2 | 24px | 1.3 | 600 | Section headers |
| H3 | 20px | 1.4 | 600 | Subsection headers |
| H4 | 18px | 1.4 | 600 | Component headers |
| Body | 16px | 1.5 | 400 | Default text |
| Small | 14px | 1.5 | 400 | Captions, labels |

## Best Practices

- Use no more than 3 levels of hierarchy on a single page
- Maintain consistent line-height for readability
- Keep line length between 50-75 characters for optimal reading
