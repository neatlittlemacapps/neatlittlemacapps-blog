# Components

Reusable UI components that make up the Infinitas Learning user interface.

## Component Library

Our component library includes:

- **Buttons** - Primary, secondary, and tertiary actions
- **Forms** - Input fields, checkboxes, radio buttons, selects
- **Cards** - Content containers with various layouts
- **Navigation** - Headers, sidebars, breadcrumbs
- **Modals** - Dialogs and overlays
- **Tables** - Data display and sorting
- **Alerts** - Status messages and notifications
