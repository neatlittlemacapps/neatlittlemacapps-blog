# Component Status

Keep track of the status of Quipu components.

This page provides an overview of all components in the Quipu design system, their current implementation status, and links to design files and Storybook examples.

## Component Readiness

Each component is tracked for:
- **Documentation Status**: New, In progress, Ready, Coming soon
- **Design File**: Link to Figma component
- **Storybook**: Link to interactive component playground

## Categories

- **Form Controls**: Button, Checkbox, Radio, Select, Textfield, etc.
- **Navigation**: Appbar, Navigation Rail, Breadcrumbs, Tabs, etc.
- **Feedback & Status**: Alert, Badge, Progress, Snackbar, Spinner, etc.
- **Content & Media**: Avatar, Card, Divider, List, etc.
- **Overlays**: Dialog, Popover, Tooltip, Side Sheet, etc.

For detailed status of all components, see the comprehensive table on the [Component Status page](https://design.infinitaslearning.com/9c11b82db/p/168ecb-component-status).
